# 🧠 戦車ゲーム開発を通じて学ぶJAVA　Python　機械学習

## 🎮 概要

- 就業支援訓練に平行して、演習としてターン制の戦車バトルゲーム（20×20マップ）を作成する
- JAVAの学習進捗に応じて内容をアップデート。最終的にはPytionを使った機械学習で敵戦車の戦闘力を高めるまでを目標とする
- JAVAの開発は現状では講義で使うeclipse上で行うが、今後はCursorやAntiGravityなどもためしていく予定
- AIを積極的に活用し、最終的には実際のコーディングはほぼAIで、それを読んで修正、または意図したコードになるように仕様を修正するような開発スタイルを確立する
- ただし、AIの活用は無料サービスの範囲にとどめる
- 今後の訓練項目のサーブレット・JSPやSQLなどの学習内容も実用性はともかく、演習として取り込めるとよいと考えてる（IUのWeb化？戦闘結果の収集？）

## 🎮 GitHubに登録する前の経過

- オブジェクト指向の学習段階に入った段階で、演習として戦車ゲームを作ることを発案
- 各戦車をクラス化し、インスタンス生成で数をふやしたり、継承により車種を増やす、機能を追加する実装を実施
- java.util.Scanner とコンソール画面によるCUI版完成（2月初旬）
- CUI版をjava.awt を使用したGUI版にアップデート（基本的にはAIに仕様を示して作成）
- ゲームの完成度上げることは目的ではないので次の段階として機械学習の導入を検討し、AIを使った調査開始
- 戦車の行動判断クラスを独立し、このクラスを強化する基本方針で、ソースを再構成
- その時期にGitHubのことを知り、AIに相談してこのプロジェクトをアップすることになった
- 上記のように学んだことをすぐ試すところから始めており、これまでの開発は無計画なので、今後は整理も平行しながら進めていく
- AIを含め、初めての経験ばかりで失敗も多く、上書き等で初期のソースや仕様書の多くが失われてしまった。
